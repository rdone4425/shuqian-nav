定时检查说明

如何启用每周自动检查

  1. 使用 Cloudflare Cron 触发器
    • 目标接口：/api/cron/weekly-check
    • 建议计划：每周日凌晨 2 点，表达式 0 2 * * 0
    • 建议同时配置授权请求头
  2. 使用外部定时任务
    • 可以通过 GitHub Actions 或其他调度器定期触发
    • 推荐使用 POST 请求并附带授权头

环境变量

  • CRON_SECRET：定时任务调用接口时使用的密钥
