导入帮助

HTML 格式

浏览器导出的标准书签文件，适合从 Chrome、Edge、Firefox 等浏览器迁移。

  • 保留基础分类结构
  • 支持标题、URL 和描述
  • 兼容主流浏览器导出文件

JSON 格式

结构化书签数据，适合程序导出或本站备份文件重新导入。

  • 支持更完整的元数据
  • 便于批量迁移
  • 适合自动化场景

注意事项

  • 单个文件建议不超过 10MB
  • 建议一次导入不超过 1000 条书签
  • 无效 URL 会被自动跳过
  • 重复书签会按导入设置处理

导入模式

  • 合并导入：保留现有书签，只导入新的条目
  • 替换导入：清空现有书签，仅保留导入内容
  • 首次迁移建议先导出一份本地备份
