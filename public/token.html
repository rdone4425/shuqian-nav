使用说明

典型流程

  1. 生成新的 API 令牌
  2. 在客户端或脚本中保存令牌
  3. 通过请求头传递令牌访问接口
  4. 不再使用时及时删除

请求头示例

X-API-Token: your-token-here

安全建议

  • 令牌具备完整 API 访问能力，请妥善保管
  • 不要在公开环境中泄露令牌
  • 建议定期轮换和清理无用令牌
  • 如果怀疑泄露，请立即删除并重新生成
